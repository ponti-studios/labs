# Context Chemistry — Product Brief

## Purpose

Make the hidden influence of context observable. This is an experimental instrument, not a general-purpose chat application.

## Core model

The user constructs an ordered context from blocks, changes one condition, and compares the resulting behavior. System instruction, prior exchange, current input, and generated continuation are different kinds of material.

## Core loop

Set a baseline, change one block or relationship, generate both outcomes, and inspect the difference.

## Novelty

It turns prompt context into a controlled experiment rather than an invisible string of instructions.

## Discovery and proof

The visitor should be able to say: “The model did not change. The context changed—and that changed the behavior.” The concept works when a first-time visitor can run one meaningful A/B test without an author-led lecture.

## Non-goals

This is not a full prompt-management suite, chat client, or token-optimization dashboard. Additional authoring features are justified only when they sharpen the causal experiment.
