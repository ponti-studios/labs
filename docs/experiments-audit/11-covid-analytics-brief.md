# COVID Analytics — Product Brief

## Purpose

Show how framing changes what becomes visible in public data.

## Core model

The same country and time series can be viewed through trends, waves, vaccination, seasonality, and outliers. Each lens poses a different question and reveals a different structure in the same underlying record.

## Core loop

Choose a country or lens, inspect the pattern, change the question, and compare what becomes visible.

## Novelty

It treats analytics as an act of interpretation rather than a race toward maximum metric density.

## Discovery and proof

The visitor should understand: “The data did not change. The lens changed—and a different story became visible.” The concept works when a visitor discovers one meaningful pattern without inspecting every chart or metric.

## Non-goals

This is not a metric warehouse, generic dashboard, or claim that visualization alone explains public health. It should prioritize interpretation over exhaustiveness.
