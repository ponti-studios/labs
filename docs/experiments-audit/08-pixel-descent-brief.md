# Pixel Descent — Product Brief

## Purpose

Create an algorithmic portrait of urban circulation in which the crowd remains legible as a collection of individual lives.

## Core model

Seeded walkers with distinct attributes move through a shared historical environment toward common infrastructure. The algorithm creates collective behavior without erasing individual variation.

## Core loop

Watch the crowd form, change the generative conditions, and observe how small parameter changes alter the collective image.

## Novelty

It uses a generative system to make mass movement feel personal without abandoning the visual language of a crowd.

## Discovery and proof

The visitor should notice: “A crowd is not noise. It is a choreography of lives.” The concept works when the artwork remains compelling without interaction and interaction deepens attention to individual-versus-collective movement.

## Non-goals

This is not primarily a p5.js technical demo or a parameter dashboard. The controls and rendering system serve the artwork.
