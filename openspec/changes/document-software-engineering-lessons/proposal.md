## Why

The `_software-engineering` curriculum contains algorithm and data structure implementations, but lacks explanatory comments. Adding comprehensive comments to every line/section explains not just what the code does, but why each step is necessary, making the lessons self-contained learning resources rather than just reference code.

## What Changes

- Add detailed comments to every JavaScript/TypeScript file in `_software-engineering/`
- Comments explain: purpose of each function/variable, algorithmic decisions, time/space complexity considerations, common pitfalls
- Comments use educational tone suitable for learning
- Preserve all existing code functionality

## Capabilities

### New Capabilities
- `documented-lessons`: All 16 lesson files in `_software-engineering/` receive comprehensive inline documentation suitable for self-study and reference

### Modified Capabilities
- None
