## ADDED Requirements

### Requirement: Curriculum README
The system SHALL provide a `_software-engineering/README.md` file that serves as the curriculum overview. This README SHALL include:
- Learning path with lesson progression order
- Prerequisites mapping between lessons
- Category groupings for related topics
- Brief description of each lesson's topics

### Requirement: Hash Tables Lesson
The system SHALL provide a `_software-engineering/13-hash-tables.ts` lesson file covering:
- Key-value storage concepts
- Hash function basics
- Collision handling (chaining, open addressing)
- Practical use cases

### Requirement: Linked Lists Lesson
The system SHALL provide a `_software-engineering/14-linked-lists.ts` lesson file covering:
- Singly linked list implementation
- Doubly linked list implementation
- Insertion, deletion, traversal operations
- Time complexity comparisons with arrays

### Requirement: BFS/DFS Lesson
The system SHALL provide a `_software-engineering/15-bfs-dfs.ts` lesson file covering:
- Breadth-first search (BFS) implementation
- Depth-first search (DFS) implementation
- Graph traversal use cases
- Stack/queue usage in traversal

### Requirement: Stack/Queue Lesson
The system SHALL provide a `_software-engineering/16-stack-queue.ts` lesson file covering:
- LIFO (stack) vs FIFO (queue)
- Array-based implementations
- Practical applications (expression evaluation, BFS)
- Common problem patterns
